onlyWarnOnMissingPropertyAccess(a)

Hides an object behind a proxy that only gives a warning if a non-existant property is accessed, not an error. This is useful for UniformProvider subclasses with dynamically added properties. If a property is removed or optimized out from GLSL code during a testing change, code setting it still works.
