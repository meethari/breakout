Class: UniformProvider

UniformProvider

A Component base class using the Composite pattern. [ProgramReflection]{ProgramReflection.html} can be used to add properties to UniformProvider-subclass-instances, reflecting uniform inputs of shader programs.

new UniformProvider(glslStructNames)

Should be called by subclass constructors. Creates a UniformProvider without any subcomponents.
Parameters:
Name Type Argument Description
glslStructNames String <repeatable>
 The names of the uniform structs in the shaders that should be reflected in ---and their values provided by--- this object.

Methods

addComponentsAndGatherUniforms(components)

Adds subcomponents and calls [addComponentsAndGatherUniforms]{addComponentsAndGatherUniforms}. [ProgramReflection]{ProgramReflection.html} instances at the leaves of the component hierarchy can then create properties matching the programs' uniforms in this object.
Parameters:
Name Type Argument Description
components UniformProvider <repeatable>
 Subcomponents to add.

definePropertiesMatchingUniforms(target)

Recursively calls definePropertiesMatchingUniforms on subcomponents. [ProgramReflection]{ProgramReflection.html} instances at the leaves of the component hierarchy can then create properties matching the programs' uniforms in the target object.
Parameters:
Name Type Description
target UniformProvider The target object.

draw()

Recursively calls draw on subcomponents, inserting all providers along the call path into the parameter list. [ProgramReflection]{ProgramReflection.html} instances at the leaves of the component hierarchy can then set uniform values from all providers in the parent chain. Other drawable objects (notably geometries) at the leaves of the node hierarchy can draw contents.
