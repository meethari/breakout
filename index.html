Breakout

What's Breakout

Breakout is an incredibly popular arcade game developed and published by Atari Inc., an originally released in 1976.

This game is a version of Breakout written in JavaScript using the WebGL Library. It is based on starter code created for the Computer Graphics class at AIT Budapest.

Controls

  • Start the game: Space
  • Move the paddles: Left/Right
  • Pause: Esc

Rules

  • Move the paddle to bounce the ball onto the bricks. Each time you don't catch the ball using your paddle, you lose a life.
  • Hitting a brick gives you points. The higher up the brick, the more points you get
  • You begin the game with 3 lives. Break all the bricks before losing your third life to win!
